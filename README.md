### 📊 Sistema de Gerenciamento de Consultores e Equipes

### 📌 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de aplicar conceitos fundamentais de Programação Orientada a Objetos (POO) em Java, simulando um sistema de gerenciamento de consultores dentro de equipes.
O sistema permite criar equipes, adicionar consultores com diferentes especialidades e níveis, além de listar e gerenciar essas informações via console.

### 🚀 Funcionalidades

✔ Criar equipes
✔ Adicionar consultores a uma equipe
✔ Listar consultores de uma equipe (ordenados por nome)
✔ Listar todas as equipes com seus respectivos consultores
✔ Remover equipes
✔ Menu interativo via terminal

### 🧠 Conceitos aplicados

✔ Herança
✔ Polimorfismo
✔ Encapsulamento
✔ Abstração
✔ Interface Comparable
✔ Estruturas de dados (List, Map)
✔ Ordenação de objetos
✔ Organização em pacotes


### 🛠 Tecnologias utilizadas

☕ Java 17+

📦 Collections Framework (ArrayList, HashMap)

🧩 Programação Orientada a Objetos
